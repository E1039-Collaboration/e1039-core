3 ## Build
4 Use [automake](https://www.gnu.org/software/automake/), [autoconf](https://www.gnu.org/software/autoconf/autoconf.html) and [libtool](https://www.gnu.org/software/libtool/) to build the code. Each package should contain 3 files: 'autogen.sh', 'configure.ac' and 'MakeFile.am'
5 
6 ```
7 mkdir [path_to_build]
8 cd [path_to_build]
9 [path_to_package]/autogen.sh --prefix=[path_to_install]
10 make -j 4
11 make install
12 ```
13 
14 Need to build using the the following order
15 - **phool**: Fun4All IO interface 'NodeTree' and IO manager
16 - **ffaobjects**: Sync multiple input streams.
17 - **fun4all**: framework related classes
18  - `Fun4AllBase`: verbosity control
19  - `Fun4AllServer`: the interface of the Fun4All framework
20  - `Fun4AllInputManage`
21  - `Fun4AllOutputManage`
22  - `SubsystemReco`
23 - **interfaces**: user interface objects
24 - **modules**: user modules
